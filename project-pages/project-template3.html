Problem Statement:
The project aims to analyze and optimize the flow of passengers on the London Subway system using a network model. The primary goal is to compute the maximum flow between stations, identify bottlenecks, and explore ways to improve subway capacity and efficiency. The problem is framed as a network flow problem with nodes representing subway stations and arcs representing direct connections between them.
Project Detail:
Part 1: We want to compute how many passengers can transit on average from the north-east part of the city center, to the south-west. To do so, compute a maximum flow between King's Cross St. Pancras (#145) and Hammersmith (#110) on the instance in the file ''connections'' (ignore the time it takes to go between stations).
Model Formulation

Model Formulation

Sets and Parameters

  • Stations: Represented as nodes in the network.
  • Connections: Represented as directed edges with capacities derived from line capacities.
  • Source Station: Station \(145\).
  • Sink Station: Station \(110\).
  • Capacity Map: Randomly assigned capacities for each line between stations.

Variables

  • \(x_{u,v}\): Continuous decision variable representing the flow from station \(u\) to \(v\), bounded by the connectionâ€™s capacity.

Objective Function

Maximize the net flow leaving the source station:

\[\max \sum_{v} \text{flow}_{\text{origin}, v} - \sum_{u} \text{flow}_{u, \text{origin}}\]

Constraints

  1. Capacity Constraint: The flow on any connection cannot exceed the capacity of that connection:

    \[\text{flow}_{u, v} \leq \text{capacity}_{u, v}\]

  2. Flow Conservation: For all intermediate nodes (excluding source and sink), the total incoming flow equals the total outgoing flow:

    \[\sum_{i} \text{flow}_{i, n} = \sum_{j} \text{flow}_{n, j}\]

Part 2: We want to understand how many people can be evacuated from the city center within 18 minutes. To do so, restrict to the stations in the file ''stations_restricted'' (and the corresponding arcs). Compute the maximum flow over time between King's Cross St. Pancras (#145) and Victoria (#273) for T=18.
Model Formulation

Model Formulation

Sets and Parameters

  • \(T\): Total time allowed for evacuation.
  • \(\text{capacity}_{u,v}\): Maximum flow allowed on the connection from station \(u\) to station \(v\).
  • \(\text{time}_{u,v}\): Time required to travel from station \(u\) to station \(v\).
  • Source Station: Station \(145\).
  • Sink Station: Station \(273\).

Variables

  • \(x_{u, v, t}\): Integer variable representing the flow from station \(u\) to station \(v\) at time \(t\), with an upper bound \(\text{capacity}_{u, v}\).

Objective Function

Maximize the total flow into the sink station (\(273\)):

\[\max \sum_{u \in \text{pred}(273)} \sum_{t=0}^{T} x_{u, 273, t}\]

Constraints

  1. Capacity Constraint:

    \[x_{u, v, t} \leq \text{capacity}_{u, v}\]

  2. Flow Conservation:

    \[\sum_{u \in \text{pred}(n)} x_{u, n, t - \text{time}_{u, n}} = \sum_{v \in \text{succ}(n)} x_{n, v, t}\]

  3. No Inflow into Source Station:

    \[\sum_{u \in \text{pred}(145)} \sum_{t=0}^{T} x_{u, 145, t} = 0\]

  4. No Outflow from Sink Station:

    \[x_{273, v, t} = 0, \quad \forall v, t\]
